# Faceit-Error
Forbidden driver:
\Device\HarddiskVolume3\Windows\System32\drivers\vmulti.sys
Please try to restart your PC. Get help with this error.

Faceit Error Driver HUION Tables
the problem is that some people have the HUION TABLE driver installed, which is a driver for a graphics tablet. There's nothing wrong with this, but FACEIT AC thinks it's a mouse interception and emulation. I'm giving you cmd through .exe, where it shows you if you have the driver and where it is. Once you realize it, you can uninstall it using Unistall.
